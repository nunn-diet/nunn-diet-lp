# 6ヶ月ToDoリスト（完全初心者向け・超詳細版）

---

## 全体の流れ

| フェーズ | 期間 | テーマ |
|---------|------|--------|
| 準備期 | WEEK 0 | 初期設計・アカウントコンセプト確定 |

## WEEK 0｜初期設計を固める（アカウント作成前に全部終わらせる）

> ここが一番つまずきやすい場所。焦らず1つずつ進めてください。

### STEP A｜自己分析シートを書く

**① 好き／得意を深掘る質問（20問）**
- [ ] 質問①〜⑩に回答する（何となく人よりうまくできること・自分では当たり前だと思っていること など）
- [ ] 質問⑪〜⑳に回答する（今の自分が当時の自分にかける言葉・乗り越えられた理由 など）

**② 本当の自分を見つける（20問）**
- [ ] 質問①〜⑩に回答する（周りに見せている自分と本当の自分の違い・自分の弱みや思い込み など）
- [ ] 質問⑪〜⑳に回答する（忘れられない言葉・本当はやりたいのに怖くて避けていること など）

**③ 全体の把握**
- [ ] 「周りからよく言われること」を記入する
- [ ] 「大切にしている価値観」を記入する
- [ ] 「無意識にできていること・強み」を記入する
- [ ] 「これまでの経験・ストーリー」を記入する
- [ ] 「夢中になれるテーマ」を記入する
- [ ] 「人より知識や経験がある分野」を記入する
- [ ] 「自分ならではの視点・特徴」を記入する
- [ ] 「サポートしたい人」を記入する

**④ 差別化**
- [ ] 「人と違うかもしれない環境」を記入する（マイナスなことでもOK）
- [ ] 「外見に関して」を記入する
- [ ] 「経験や夢」を記入する
- [ ] 「人間関係」を記入する
- [ ] 「お金」を記入する

---

### STEP A'｜ぬんAIでアカウントの方向性を相談する

> 自己分析が終わった段階でいったんAIに相談。「どんなアカウントにするか」の方向性を固める。

- [ ] ぬんAIを開いて自己分析の結果を全部伝える
- [ ] 以下の観点で相談する
  　・**プロセスエコノミー型**か（今の状態→ゴールまでの過程を発信）
  　・**価値観・情報発信型**か（自分が持っている知識・ノウハウを発信）
  　・あるいは2つを組み合わせる形か
- [ ] AIが出してくれた候補をもとに、自分の方向性を仮決定する
- [ ] 仮決定した内容をメモしておく（次のSTEP Bで使う）

---

### STEP B｜ジャンルを決める（大→中→小）

> 大ジャンルだけで発信するのは危険。幅が広すぎるコンテンツには需要がない。必ず小ジャンルまで絞り込む。

- [ ] 大ジャンルを決める（例：料理・ダイエット・美容 など）
- [ ] 中ジャンルを決める（例：大ジャンルが料理なら → レシピ）
- [ ] 小ジャンルを決める（例：中ジャンルがレシピなら → ダイエット・ヘルシー・痩せさせるレシピ）
- [ ] 決定した小ジャンルをメモする（これが発信の軸になる）

---

### STEP C｜モデリング分析シートを書く（10アカウント以上）

> 「このアカウント良いな」で終わらず「なぜ良いのか」「どうやって収益を作っているのか」まで言語化するのが目的。

- [ ] Instagramアプリをインストールする（アカウントはまだ作らない）
- [ ] 決定した小ジャンルに当てはまる競合アカウントを10個以上ピックアップする
- [ ] 「関連ジャンル × 同じ発信タイプ」の競合も探す
- [ ] モデリング分析シートに10アカウント分を記入する（21項目）
- [ ] 10アカウント分を書き終えたら、伸びている投稿の共通点を3つ書き出す
- [ ] 「自分が真似できそうな要素」と「差別化できるポイント」をまとめる
- [ ] モデリング先を1〜2アカウントに絞る

---

### STEP D｜競合分析（差別化ポイントを見つける）

- [ ] 「すでにいるアカウント」と「まだいないアカウント」を書き出す
  　例：痩せるレシピのアカウントは多いが、夫を痩せさせるレシピは少ない → そこが差別化になる
- [ ] 差別化方法を考える（数字で勝負するのではなく「ベクトルを変える」「言葉の表現を変える」）
- [ ] 差別化されたアカウントコンセプト（1文）を仮決定する

---

### STEP E｜ペルソナ設計

> スマホの向こうにいる「具体的な1人」をイメージできないと、どんな投稿をすればいいかわからなくなる。

- [ ] 性別・年齢・居住地・職業・年収・家族構成を書く
- [ ] 日常の生活・趣味・悩みを書く
- [ ] 「この人の感情を動かすにはどんな投稿が必要か」を3つ書き出す
- [ ] ペルソナ設計を確定する

---

### STEP F｜プロフィール名の方向性だけ考える

> 詳しいプロフィール文の作成は1ヶ月目WEEK 1で行う。ここでは「どんな名前にするか」の方向性だけ決める。

- [ ] 個性・権威性・提供できる情報を箇条書きで書き出す
- [ ] プロフィールAIアプリ（https://nunn-diet-lp.vercel.app/insta-profile-ai/）を使ってプロフィール名の候補を出してもらう
- [ ] AIが出した候補をもとに、より魅力的な名前に磨く
- [ ] プロフィール名を1つに仮決定する（1ヶ月目WEEK 1で確定させる）

---

### STEP G｜キャラクター設定を決める

- [ ] 憧れ（　）% ／ 共感（　）% ／ 応援（　）% の割合を考える
- [ ] 自分のキャラを1行で表現する（プロフには書かない・自分の軸として持つ）
  　例：「明るくて見てると笑える人」
- [ ] 「やること」リストを作る（3〜5個）
  　例：素直でいること・プライドを感じさせないこと
- [ ] 「やらないこと」リストを作る（5〜10個）
  　例：もやもやをそのまま発信しない・特定の商品をディスらない・偉そうにしない
  　※やらないことの方が多くなるのが正常

### STEP H｜ぬんAIと最終壁打ち → ぬんと個別セッション

- [ ] ぬんAIにジャンル・コンセプト・ペルソナ・キャラ設定の全内容を共有する
- [ ] 「どんなアカウントにするか」「強み・弱み」をAIと一緒に深掘りする

## 2ヶ月目｜資格取得・本番リール・モニター準備

### WEEK 5〜6｜本番リールを投稿する

> 練習リールで身につけたスキルを使って、いよいよ本番投稿を始める。

**初投稿の流れ（毎回この順番で）**
1. テーマを決める → 台本を書く（冒頭1秒のメンタルモデルを必ず意識）
2. 保存される要素を1つ以上入れる（再現性・有益性・複雑性・網羅性）
3. 素材を撮影する → CapCutで編集 → Canvaでカバー画像を作る
4. **まずTikTokに投稿する**
5. 12〜24時間後に反応を確認する
6. 冒頭1〜3秒で離脱が多ければ、冒頭を編集して再投稿する
7. **以下の条件を満たしてからInstagramに投稿する**
   　・5秒以上の視聴維持率が50%以上
   　・TikTokでの24時間再生回数が1,000回以上

- [ ] Day 29｜1本目のリールのテーマを決めて台本を書く
- [ ] Day 30｜素材を撮影する
- [ ] Day 31｜CapCutで編集 → Canvaでカバー画像作成
- [ ] Day 32｜TikTokに投稿する（Instagramにはまだ上げない）
- [ ] Day 33｜12〜24時間後に反応を確認する
  　（冒頭1〜3秒で離脱が多い場合は冒頭を編集 → 再投稿）
- [ ] Day 34｜5秒維持率50%・24時間1,000再生を超えたらInstagramにも投稿する
- [ ] Day 35｜2本目のリールを同じ流れで制作・TikTok投稿 → 確認 → Instagram投稿
- [ ] Day 36｜ストーリーズを毎日更新し続ける（以降ずっと継続）
- [ ] Day 37｜ストーリーズ：「資格の勉強を始めました」匂わせ投稿
- [ ] Day 38｜3本目のリールを制作・TikTok投稿 → 確認 → Instagram投稿
- [ ] Day 39｜ストーリーズ：「取れたらどんな人を痩せさせたいか」を発信
- [ ] Day 40｜4本目のリールを制作・TikTok投稿 → 確認 → Instagram投稿

## 3ヶ月目｜価格を引き上げる

### 毎週のリール投稿（週2本）

- [ ] テーマを決める（競合の伸びている投稿をリサーチ）
- [ ] 台本を書く（冒頭1秒のメンタルモデルを意識）
- [ ] 保存される要素を1つ以上入れる（再現性・有益性・複雑性・網羅性）
- [ ] 素材を撮影 → CapCutで編集 → Canvaでカバー作成
- [ ] まずTikTokに投稿 → 反応確認（維持率50%・1,000再生）→ Instagramに投稿

### 毎日のストーリーズ（ファン化3要素を意識）

- [ ] 毎日更新し続ける
- [ ] 憧れ要素：実績・専門知識をさりげなく出す
- [ ] 共感要素：失敗談・本音
- [ ] 応援要素：挑戦の様子・進捗報告

### モニター2人目以降の価格引き上げ

- [ ] モニター2人目の募集金額を決める（酵素ドリンク代より少し上げる）
- [ ] 「興味あり」リストの方に個別DMを送る → LINE → Zoom調整
- [ ] Zoomでクロージング → モニター2人目のサポート開始
- [ ] モニターのビフォーアフターをコンテンツ化する（許可取り必須）
- [ ] 今月の新規クライアント数を確認する（目標：月2名）

### 毎月の振り返り・キャラ確認

- [ ] 競合アカウントのバズ投稿を追加分析する
- [ ] 自分のファン化3要素の割合を確認する
- [ ] 「やらないことリスト」を守れているか確認する
- [ ] プロセス型の場合：最初のバズリールと現在のストーリーで人格がずれていないか確認する
- [ ] 今月うまくいったこと・改善点をメモする
