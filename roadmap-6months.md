# 6ヶ月ロードマップ

## 全体の流れ

| フェーズ | 期間 | テーマ |
|---------|------|--------|
| 準備期 | 1〜2ヶ月目 | 土台づくり＋資格取得 |
| 助走期 | 3ヶ月目〜 | 価格を引き上げながら継続 |
| 加速期 | 4〜6ヶ月目 | 月2名ペースで積み上げ |

---

## 発信ルール（全期間共通）

| 項目 | ルール |
|------|--------|
| リール | 週2本 |
| ストーリーズ | 毎日（基本） |
| 集客フロー | ストーリーズ → DM → LINE誘導 → Zoomクロージング |
| 獲得目標 | 月2名の新規クライアント |

---

### 2ヶ月目｜モニターを集め、サポートを開始する

#### インスタ（メイン）

**ストーリーズの流れ（資格取得に向けて）**
- [ ] 資格の「匂わせ」を1週間ほどストーリーズでアップ
  （資格が取れたら、どんな人を痩せさせたいか、自分の思いも発信）
- [ ] 資格試験の様子を2日ほどアップ
  （合否のドキドキ期間もフォロワーと一緒に体験してもらう）
- [ ] 合格したら全力でお祝い投稿！ストーリーズで盛大に発信
- [ ] 合格後：ダイエットモニター募集開始
- [ ] ストーリーズのスタンプ or 公式LINEで「モニターに興味あり」リストを作成
- [ ] DMでフォロワーとのコミュニケーションを継続強化
- [ ] モニター1人目：酵素ドリンク代のみでサポート開始
- [ ] モニターのビフォーアフターをコンテンツ化（※許可取り必須）
- [ ] 個別カウンセリング用スライドの完成
  （土台はできているので、自分のストーリーを盛り込んで仕上げる）

#### 集客フロー（固定）
1. ストーリーズで興味あり反応を集める
2. DM → LINE誘導
3. ZoomでZoomクロージング

#### 資格（サブ）
- [ ] 1ヶ月半〜2ヶ月目の間に取得

---

### 3ヶ月目〜｜価格を引き上げる

- [ ] 2人目以降：準備ができ次第、少しずつ価格を引き上げながら募集
- [ ] 集客フロー継続（ストーリーズ → DM → LINE → Zoom）
- [ ] モニターの成果をコンテンツ化し続ける（ビフォーアフター・体験談）
- [ ] **推奨目標：月2名の新規クライアント獲得**

---

### 4〜5ヶ月目｜仕組みを回す

- [ ] 月2名ペースを維持しながら、集客〜クロージングの型を固める
- [ ] 実績が増えてきたタイミングでさらに価格を調整
- [ ] 既存クライアントの成果をコンテンツ化し続ける

#### モニター候補リストの運用
- [ ] ストーリーズのスタンプ or 公式LINEでリストに追加し続ける
- [ ] リストの方へ個別DMを送ってアプローチ
- [ ] リストが尽きたらストーリーズで再募集

- [ ] **推奨目標：月2名の新規クライアント獲得**

---

### 6ヶ月目｜自走できる状態へ

- [ ] 集客 → 相談 → 成約のサイクルが自分で回せている
- [ ] 単価をさらに引き上げ、収益を安定させる
- [ ] 6ヶ月間の振り返り（効いたこと・改善点の棚卸し）
- [ ] 次のステップを検討（グループ講座・単価アップ・紹介経由の集客など）
- [ ] **推奨目標：月2名の新規クライアント獲得**

---

## 価格の段階イメージ

| 時期 | クライアント | 価格感 |
|------|-------------|--------|
| 2ヶ月目・1人目 | モニター1号 | 酵素ドリンク代のみ |
| 2人目以降（準備ができ次第） | モニター2号〜 | 少しずつ引き上げ開始 |
| 4〜6ヶ月目 | 継続獲得 | 段階的に調整 |
